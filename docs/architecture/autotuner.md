# Auto-Tuner

Search Strategies
- Grid/random; Bayesian optimization; genetic algorithms; reinforcement learning.

Cost Models
- XGBoost, GNNs, ensembles; uncertainty estimates for exploration/exploitation trade-offs.

Features
- Graph structure, op types, hardware descriptors, profiling signals.
- Transfer learning/warm-starting across hardware generations.

Interfaces
- Tunable configuration schema.
- Pluggable cost model and search strategy interfaces.
